## Project Description

The ScoreTrakker project is a Java program designed to read and process student scores from text files. It offers functionality for displaying the student records in a sorted order based on their names. 
## Project Structure

The project consists of the following classes:

- ScoreTrakker class: Contains an ArrayList to store student records. It provides methods for loading data from a file and printing the data in sorted order.
- Student class: Represents individual students with attributes for their names and scores. It implements the Comparable interface to enable sorting.
